## 4. In-App Behaviour Must Match the Strings

Apple reviewers actively test this.

Make sure:

* The photo permission prompt appears **only when the user initiates a scan**
* No photo access happens silently on app launch
* Saving back to the library only happens after a clear user action

If your app auto-starts analysis on first launch, add a **pre-permission screen** explaining why.
